# 🌸 7 лет нашей дружбы - Романтичный сайт-подарок

Красивый и нежный сайт-подарок для девушки, посвящённый 7 годам дружбы. Сайт содержит 5 интерактивных страниц с фотографиями, тестом на воспоминания, галереей артов и секретной финальной страницей.

## 🚀 Как загрузить сайт на GitHub Pages

### Шаг 1: Создайте репозиторий на GitHub
1. Зайдите на [GitHub](https://github.com) и войдите в свой аккаунт
2. Нажмите "New repository"
3. Назовите репозиторий (например: `7-let-druzhby`)
4. Выберите "Public" (обязательно для GitHub Pages)
5. Нажмите "Create repository"

### Шаг 2: Загрузите файлы в репозиторий
**Способ 1: Через веб-интерфейс**
1. Нажмите "Add file" → "Upload files"
2. Перетащите все файлы проекта (включая папку `images`)
3. Напишите сообщение коммита (например: "Initial commit")
4. Нажмите "Commit changes"

**Способ 2: Через Git (если установлен)**
```bash
git clone https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
cd НАЗВАНИЕ_РЕПОЗИТОРИЯ
# Скопируйте все файлы проекта в эту папку
git add .
git commit -m "Initial commit"
git push origin main
```

### Шаг 3: Активируйте GitHub Pages
1. В вашем репозитории перейдите в "Settings"
2. В меню слева выберите "Pages"
3. В разделе "Build and deployment" выбуйте "Deploy from a branch"
4. В "Branch" выберите `main` и папку `/root`
5. Нажмите "Save"

### Шаг 4: Дождитесь публикации
Через 1-2 минуты сайт будет доступен по адресу:
`https://ВАШ_НИК.github.io/НАЗВАНИЕ_РЕПОЗИТОРИЯ/`
